Available placeholders 🖨 Print this reference

Type these inside curly braces in your subject or body. They get replaced with the donor's real info when the email actually sends:

  • {first_name} — donor's first name (e.g., Sarah)
  • {last_name} — donor's last name (e.g., Freidenberg)
  • {name} — full name (e.g., Sarah Freidenberg)
  • {greeting} — full greeting line: Dear Sarah in English, שלום וברכה Sarah Freidenberg in Hebrew
  • {amount} — pledged-amount line, only when you set a preset amount (renders as Your pledge: $36 / הסכום שהתחייבת: ₪100)
  • {email} — donor's email address
  • {link} — the donation URL (the donate button below the message is added automatically — only use this if you want it inline too)

Anything you don't have data for collapses to empty — the placeholder won't show up as raw text in the donor's inbox.
