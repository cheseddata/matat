# מדריך למשתמש — מתת ‎|‎ ZTorm ‎|‎ גמ״ח

> **מצב חול חוליה (Sandbox Mode)** — המערכת פועלת עם הנתונים האמיתיים שלך  
> אבל **אף חיוב, אימייל או SMS לא נשלחים באמת**. זה לבדיקה בלבד.

---

## התקנה חד־פעמית

1. ודא שמותקן Python 3.12 או חדש יותר ([הורדה מ־python.org](https://www.python.org/downloads/))  
   חשוב: במהלך ההתקנה של Python, סמן את התיבה **"Add Python to PATH"**.

2. לחיצה כפולה על **`install.bat`** בתיקיית `F:\matat_git`.  
   ההתקנה לוקחת כ־2 דקות ויוצרת קיצור־דרך בשולחן העבודה: **"Matat (Sandbox)"**.

---

## הפעלה יומית

- לחיצה כפולה על הקיצור **"Matat (Sandbox)"** בשולחן העבודה  
  (או על `start.bat` בתיקיית `F:\matat_git`).
- הדפדפן ייפתח אוטומטית לכתובת `http://localhost:5050/login`
- **שם משתמש:** `admin`   **סיסמה:** `admin123`

החלון השחור שנפתח = השרת. **אל תסגרי אותו** עד שסיימת את העבודה.  
כדי לעצור את המערכת: סגרי את החלון השחור.

## תפריט ראשי

- **מתת** (Matat) — ניהול תרומות, תורמים, קבלות
- **🏛 ZTorm** — מערכת התרומות הישנה (כולל כל הנתונים שהיו באקסס)
- **💚 גמ״ח** — קרן הגמ״ח (חברים, הו״ק, תנועות)

כפתורי התחלפה בין המערכות נמצאים בפינה הימנית־עליונה.

## שאלות נפוצות

**אני לא רואה את הנתונים שלי**  
ודאי שאת מחוברת כ־`admin`. אם כן, בדקי שמספר הכרטיס שאת מחפשת נכון.

**קיבלתי הודעת שגיאה**  
סגרי את החלון השחור, ופתחי את `start.bat` שוב. הנתונים לא נמחקים.

**אני רוצה להוסיף חבר חדש / עריכה**  
במצב ה־Sandbox ניתן להוסיף ולערוך, אבל השינויים *נשמרים מקומית בלבד*.  
אחרי אישור והעלאה לשרת, השינויים האמיתיים יתבצעו שם.

**איפה הנתונים נשמרים?**  
כל הנתונים נמצאים ב־`F:\matat_git\instance\matat.db`.  
זה קובץ SQLite שאפשר לגבות (העתקה פשוטה של הקובץ).

## מצב בדיקה ‎vs.‎ מצב חי

- **עכשיו (מצב בדיקה / Sandbox):** פס צהוב בראש כל עמוד. אין חיובי אשראי אמיתיים.
- **אחרי אישור:** המערכת תעבור לשרת matatmordechai.org והמצב יהפוך לחי.  
  רק אז החיובים, הקבלות והדוא״ל יהיו אמיתיים.
